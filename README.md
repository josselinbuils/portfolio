## Contributing

Group and compress DICOM files:

```
export DIRNAME=dirname
COPYFILE_DISABLE=1 tar --exclude='.DS_Store' -cf $DIRNAME.tar $DIRNAME && COPYFILE_DISABLE=1 tar --exclude='.DS_Store' -czf $DIRNAME.tar.gz $DIRNAME
```
